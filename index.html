Kali-2020.1更新汉化配置

2020-02-18
Kali

开始操作

首先了解一下apt和apt-get之间的区别

https://www.sysgeek.cn/apt-vs-apt-get/

感兴趣的话可以深入探究

One

apt update

这个操作并没有开始更新软件，类似于将远程源中的最新版本信息更新到本地

接下来才开始更新软件：

apt upgrade

更新系统：

apt full-upgrade

或者：

apt-get dist-upgrade

清理安装包：

apt-get clean
apt-get autoclean

Two

推荐使用

apt-get update && apt-get upgrade && apt-get clean
